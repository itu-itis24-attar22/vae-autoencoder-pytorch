# Variational Autoencoder (VAE) & Autoencoder

## Project Overview
This project explores Generative Deep Learning by implementing two types of autoencoders to reconstruct and generate MNIST digits.

## Key Features
- **Standard Autoencoder:** Built from scratch using **NumPy** with manual forward/backward propagation.
- **Variational Autoencoder (VAE):** Implemented using **PyTorch**, utilizing the reparameterization trick and KL Divergence loss.
- **Results:** Comparison of reconstruction quality and latent space generation.
